# html-lite-capture

Firefox extension : Select blocks on page & copy simplified HTML

**Repository:** [https://github.com/Pierre-LHOSTE/html-lite-capture](https://github.com/Pierre-LHOSTE/html-lite-capture)

# Build Instructions

## Installation & Build

Install dependencies:
```bash
npm install
```

Compile the extension:
```bash
npm run build
```

**Output:** The compiled extension files will be in the `dist/` directory.

Generate zip:
```bash
npm run zip
```
